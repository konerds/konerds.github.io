Java

본 자바 섹션에서는 여러 자바 강의 및 바이블 서적의 내용들을 정리하는 것을 목표로 합니다.

Why

자바는 이식성, 안정성 및 다양성을 통해 프로그래밍 언어 세계에서 여전히 확고한 위치를 유지하고 있고, 계속해서 변화하며 발전하고 있기에 개발자들 사이에서 가장 선호되는 언어 중 하나입니다.

특히 한국의 경우, 자바는 국민 언어라고 봐도 무방할 정도로 유명합니다.

Comment

번역하고 요약한 내용을 통해 자바의 기초를 다질 수 있는 계기가 되었으면 합니다.

시리즈의 내용들은 초안으로 유지되지 않으며 시간이 지남에 따라 지속적으로 수정 및 개선될 수 있습니다.

포스팅 주기는 한 주, 어쩌면 한 달이 될 수 있겠지만, 조금씩 조금씩 완성해나갈 예정입니다.


#1 - Java 환경 소개
December 3, 2023

Java 는 세계에서 가장 널리 사용되는 프로그래밍 언어 중 하나로, 비즈니스 및 엔터프라이즈 컴퓨팅에서 탁월한 성공을 거두었습니다. 본 섹션에서는 Java 언어와 JVM(Java Virtual Machine) 의 역사와 Java 생태계를 알아봅니다. 그리고 Java 와 다른 환경의 차이점에 대한 일반적인 질문과 Java 보안과 관련된 몇 가지 측면에 대해 논의합니다.

#2 - Java 문법
December 5, 2023

이 섹션은 Java 문법에 대해 간결하되 포괄적으로 알아봅니다. 가장 낮은 수준의 Java 문법에서 시작하여 점점 더 높은 수준의 구조를 다루면서 Java 문법을 알아봅니다. Java 프로그램을 작성하는 데 사용되는 문자와 해당 문자의 인코딩 Java 프로그램을 구성하는 리터럴 값, 식별자 및 기타 토큰 Java 로 다룰 수 있는 데이터 타입 Java 에서 개별 토큰을 더 큰 표현식으로 그룹화하는 데 사용되는 연산자 표현식 및 기타 명령문을 그룹화하여 Java 코드의 논리적 청크를 형성하는 명령문 다른 Java 코드에서 호출할 수 있는 명명된 메서드.
