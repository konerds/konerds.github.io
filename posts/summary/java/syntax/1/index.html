하향식으로 Java 프로그램 개요 살펴보기

Java 문법에 대한 상향식 탐색을 시작하기 전에 잠시 하향식으로 Java 프로그램을 살펴보겠습니다.

Java 프로그램은 하나 이상의 Java 소스 코드 파일 또는 컴파일 단위로 구성됩니다. 이 섹션의 마지막 부분에서는 Java 파일의 구조를 설명하고 Java 프로그램을 컴파일하고 실행하는 방법을 설명합니다.

각 컴파일 단위는 선택적 package 선언으로 시작하고 그 뒤에 0개 이상의 import 선언이 옵니다. 이러한 선언은 컴파일 단위가 이름을 정의할 네임스페이스와 컴파일 단위가 이름을 가져오는 네임스페이스를 지정합니다. 선택적 package 및 import 선언 뒤에는 0개 이상의 클래스 와 인터페이스 정의 등 참조 타입 정의가 옵니다.

참조 타입 정의를 살펴보면 필드 , 메서드 , 생성자 등의 요소들을 볼 수 있습니다. 이러한 요소들 중에서도 문장 으로 구성된 Java 코드 블록을 의미하는 메서드 는 특히 중요합니다.

위처럼 기본 용어들을 정의한 후, 어휘 토큰이라고도 불리는 문법의 기본적인 단위를 검토하여 상향식으로 Java 프로그램을 살펴보겠습니다.
