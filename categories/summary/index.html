Understanding Java 1990년대 후반부터 프로그래밍 세계의 충실한 역할을 해 온 Java는 지금까지도 비즈니스 및 엔터프라이즈 컴퓨팅 분야의 강자로 자리매김하고 있습니다. 가독성과 학습 용이성으로 잘 알려진 Java 는 C++ 와 같은 언어에 대한 업계 경험을 바탕으로 구축되었으며, 보다 더 객체지향적으로 설계되었습니다. 1996년 이후의 보수적인 발전은 안정적인 기반을 제공해왔으며, Java 8 버전부터는 이전 버전과의 호환성을 유지하면서 기존 LISP 와 같은 언어에서 람다 표현식과 같은 기능을 차용하여 최신 프로그래밍 패러다임을 수용하는 Java 의 행보를 보여줍니다.

Overview 본 자바 짜깁기 시리즈에서는 다양한 자바 강의 및 관련 서적의 내용들을 정리하는 것을 목표로 합니다. 여러 소스를 통해 자바에 대한 전체적인 관점을 학습하고 정리해볼 예정입니다. Why 자바는 이식성, 안정성 및 다양성을 통해 프로그래밍 언어 세계에서 여전히 확고한 위치를 유지하고 있고 심지어 계속해서 변화하며 발전하고 있기에 개발자들 사이에서 가장 선호되는 언어 중 하나라고 생각합니다. 번역하고 요약한 내용을 통해 자바의 기초를 다질 수 있는 계기가 되었으면 합니다. Comment 시리즈의 내용들은 초안으로 유지되지 않으며 시간이 지남에 따라 지속적으로 수정 및 개선될 수 있습니다.
